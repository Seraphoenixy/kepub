在看到灵梦一招解决掉幽玄魔眼之后，少女一度认为幽玄魔眼真是一个弱爆了的角色。

不过在跟着灵梦他们在魔界继续在魔界走了一段路之后，少女知道自己错了，而且错了离谱。

“啊咤！”

连有名字的招式都没有用，灵梦只是随手扔出了几张灵符，就轻松地炸飞了一片挡在路上的杂鱼…

好吧，其实这些小兵即使是杂鱼，但也有十多级的程度，真要算起来，可能少女连这些杂鱼都不如。

【系统提示：由于未知错误，经验插件出现损坏，您无法获得经验。】

【系统提示：团队杀死魔界人，你获得了经验310】

【系统提示：由于未知错误，经验插件出现损坏，您无法获得经验。】

…

一路上听着系统不断发出的提示声，少女感觉到很不爽。

要不是不能储存经验值，现在自己怎么说也有20多级了吧。

看着灵梦像割草一样在路上四处乱扔灵符，少女有些好奇地对着载着自己和灵梦的玄爷问道：

“玄爷，灵梦为什么要跑到魔界来捣乱啊？”

思考了一下，或许是玄爷觉得告诉少女也没什么关系，于是便把大致原因说了一下。

博丽巫女是一种传承之力，其实在很久以前它并不叫博丽，不过在博丽神社被阴阳师们的结界隔断之后，为了重新传承，这才改名为博丽神社。

博丽灵梦是第一代博丽巫女。

每一个成为博丽巫女的人都会得到一种只有她们才能够掌握的灵力，这种灵力有一部分被储存在了巫女的身体里，用来强化她的身体素质和使用各种秘法。

不过，除了巫女身体里的那些灵力，大部分的灵力则是被封印在了用于传承的博丽阴阳玉里面。

不过当灵梦试图驱动这个博丽阴阳玉时，却发现自己根本无法操纵它。

为了知道阴阳玉的使用方法，灵梦决定去魔界和地狱寻找曾经认识这个阴阳玉的人，问出操纵阴阳玉的细节。

听完了玄爷解释，少女不禁把目光放在了神玉的身上：

“那神玉是怎么回事？他应该也是阴阳玉吧。”

“他的确是阴阳玉，而且是守护博丽阴阳玉的存在，”玄爷顿了顿，继续说道，“灵梦为了找到博丽阴阳玉，在神社里跟他打了一架，然后把他打败了。”

“原来如此。”

少女点了点头，看向了灵梦，“那么灵梦你已经知道要去找谁了吗？”

“不知道。”

灵梦理直气壮地回答。

少女愣住了。

“因为博丽神社里并没有留下多少文献，所以我们也仅仅知道在魔界和地狱有知道博丽阴阳玉的人而已，但具体是谁我们还不清楚。”

说完，玄爷叹了口气：

“本来我是劝灵梦先调查一下再来的，可是灵梦为了早点使用阴阳玉，所以急匆匆地就…”

“啰嗦！”

灵梦脸红红地打断了玄爷的话，“我还不是为了能早点使用阴阳玉！”

看了看傲娇的灵梦和无奈的玄爷，少女在心中对玄爷表示了崇高的敬意。

（照顾这么任性的灵梦，真是辛苦你了，玄爷。）

=============================

“话说，你们没发现周围的人又变少了吗？”

神玉突然出声提醒了一下众人，这才让少女发现灵梦不知什么时候已经停止了乱丢灵符的行为，周围也变得一片寂静。

“之前遇到幽玄魔眼的时候似乎也是这个样子，难道又有人要出来了吗？”

玄爷也是谨慎地放慢了脚步——这位又会飞又会跑的乌龟爷爷现在看起来总算有点乌龟的样子了。

“哦？又有人来了？很好，希望这次能遇到一个会说话的角色，不要像上次那些眼睛连话都不说。”

灵梦兴奋地从衣袖里掏出一把灵符——话说你一路上丢了那么多灵符居然还没丢完吗！？

“其实我一直想吐槽这种BOSS出来之前小兵要全部回避的设定来着…”

少女则是在说些其他人听不懂的话。

果不其然，没等玄爷走多久，就看到一个红色的身影站在前方。

那是一个拥有紫色的瞳孔和金色的头发的少女，头上绑着红色的大发带，左手持有带着白色星形顶端的法杖。

服装看上去很艳丽，在白色衬衣外穿着的是带有大红丝带的蓝色背心和镶有紫色边缘的红长裙。

少女看了看这个人的头顶，几个跟她裙子颜色很像的鲜红色大字浮在她的头上。

【伊莉丝（LV318）

称号：无邪气的恶魔

查看资料邀请组队申请交易】

（啊咧？居然是有称号的角色？）

少女愣了一下，看了看身边的灵梦，轻声道：

“这个人叫伊莉丝，无邪气的恶魔。”

“诶？这个人就是伊莉丝！？”

出乎意料的，灵梦似乎认识这个人。

“伊莉丝是最不像恶魔的恶魔，她应该是向来讨厌打架的，既然她会出现在这里，只能说明是她的主人让她来的。”玄爷沉吟了一会儿对着少女解释了一下。

“嗯…”

少女点点头，心念一动打开了伊莉丝的资料。

【人物名称：伊莉丝

性别：女

种族：恶魔

力量：1998（肉体强度和身体恢复力）

敏捷：1996（行动速度和身体柔韧度）

神秘：52800（精神力量和特殊能力）

运气：109（闪避几率和掉宝率）

天赋：血裔（能化身成蝙蝠，拥有通过吸血补充体力的能力）；魔法（能使用金﹑木﹑土﹑月四曜魔法）；魔性（未知能力，无法解读）】

（无法解读！？）

少女心中一惊，对着灵梦小声提醒道：

“小心点，这个人有点古怪。”

“我知道。”

灵梦拿出灵符对着伊莉丝，表情严肃。

“啊…真是的，为什么非得战斗不可啊…”

看到灵梦已经做好了战斗准备，伊莉丝的脸一下就垮了下来。

“我来这里并不是为了惹事，我只是想问点关于阴阳玉的事情而已，希望你不要阻止我。”

灵梦面无表情地对着伊莉丝开口道。

“哈…？阴阳玉？”伊莉丝看着灵梦，“那个红白色的八卦球？”

“你知道！？”灵梦惊喜地看着伊莉丝，“你知道怎么用吗？”

“我又没用过，怎么会知道。”

伊莉丝摊开手，“也许我主人知道…”

“很好！快带我去找你主人吧！”

灵梦兴奋地打断了伊莉丝的话，那表情就像是一个酒鬼找到了一坛陈年老窖一般。

“带你们去找主人…？”伊莉丝呆了一下，“不行不行！要是带你们去的话…”

说完，伊莉丝仿佛想到了什么很可怕的事情一般，满脸惊恐地捂住脸，慢慢地倒在了地上。

看到伊莉丝倒地，神玉飞了过去，在伊莉丝上空转了一会儿，然后又飞了回来：

“晕倒了…”
